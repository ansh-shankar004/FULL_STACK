Introduction

Project Overview

This project is a comprehensive frontend implementation for "Kuber Classic," a luxury family restaurant. We have developed a modern, single-page website designed to serve as the restaurant's primary digital touchpoint, reflecting its brand identity of elegance and quality. The site provides essential functionalities wrapped in a visually appealing and responsive user interface.

Objective

Our primary objective was to engineer a high-fidelity web presence that not only attracts customers but also provides practical tools for engagement. This includes creating an immersive user experience for browsing, enabling seamless table reservations, and offering an instant event cost estimation tool, thereby enhancing customer convenience and streamlining restaurant operations.

Website Overview (Phase 1)

We have successfully developed a complete, single-page static website that captures the essence of the Kuber Classic brand with a focus on aesthetics, UX, and responsiveness.

Feature: Hero Image Slider

  • Opens with a full-screen, dynamic hero section.
  • Carousel of high-quality, fading images creates a luxurious feel.
  • Prominent "Reserve Your Table" CTA button.
  • Subtle scroll-down arrow encourages exploration.

Feature: Navigation & Sections

  • Sticky Navigation Bar: Remains visible on scroll for easy access.
  • Smooth Scrolling: Clicking a link smoothly navigates to the section.
  • Mobile Responsive Menu: Collapses into a clean hamburger menu on small devices.

Features: Reservation & Quote Forms

  • Table Reservation: User-friendly form with dynamic time slot generation and validation.
  • Event Quote Estimator: Interactive tool for instant event cost estimation with conditional discount logic.
  • Data Handling: Both forms compile data into JSON objects, simulating a backend submission.

Technology Stack – Phase 1

  • HTML5: Used for the semantic structure and content of the website.
  • Tailwind CSS: A utility-first CSS framework for rapid, responsive UI development.
  • Vanilla JavaScript: Powers all client-side interactivity, form logic, and DOM manipulation.
  • Google Fonts: Leveraged for typography (Playfair Display & Inter) to match the luxury theme.

Phase 2 – Planned Features

The next step is to transform the static site into a full-stack, data-driven web application. This involves adding a backend, database, user authentication, and advanced interactive features.

Planned Features

1. Menu & Cart System

  • Interactive menu with item details.
  • Persistent shopping cart & payment gateway.

2. Advanced Reservation System

  • Backend integration with real-time availability.
  • Automated email/SMS confirmation.

3. User & Admin Login

  • Secure authentication with JWT & bcrypt.
  • Role-based dashboards for users and admins.

4. Customer Feedback System

  • Dedicated form for users to submit reviews.
  • Admin can view and manage all feedback.

5. Chatbot Integration

  • 24/7 AI-powered support for FAQs and booking assistance.

Technology Stack – Phase 2

  • Frontend: React.js + Tailwind CSS
  • Backend: Node.js + Express.js
  • Database: MongoDB (NoSQL)
  • Authentication: JWT, bcrypt
  • AI / ML: Python (Flask + NLP Libraries) or OpenAI API
  • Payments: Razorpay or Stripe
  • Deployment: Netlify (Frontend), Render/Vercel (Backend)
